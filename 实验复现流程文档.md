```
┌═══════════════════════════════════════════════════════════════════════════════════┐

│                     目标: 学习类别 0-9 (CIFAR-100示例)                              │
│                     训练参数: Adapter参数 (约1-2M)                                   │
│                     冻结参数: ViT主干 (约86M)                                       │
└═══════════════════════════════════════════════════════════════════════════════════┘

输入: 图像 [B, 3, 224, 224]
  │
  │  例如: B=48, 224×224 RGB图像
  │
  ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│                    预训练 ViT-B/16 + Adapter                                 │
│                    (SimpleVitNet_AL 初始化)                                  │
│  ┌────────────────────────────────────────────────────────────────────┐    │
│  │  1. Patch Embedding                                                │    │
│  │     - 16×16 patches → 768维向量                                     │    │
│  │     - 输出: [B, 196, 768] (14×14个patches)                          │    │
│  │                                                                     │    │
│  │  2. Position Embedding + CLS Token                                 │    │
│  │     - 加入可学习位置编码                                             │    │
│  │     - 输出: [B, 197, 768] (196 patches + 1 CLS)                     │    │
│  │                                                                     │    │
│  │  3. 12× Transformer Blocks (带 Adapter)                            │    │
│  │     ┌──────────────────────────────────────────┐                   │    │
│  │     │  Block i (i=0..11):                      │                   │    │
│  │     │                                          │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ LayerNorm               │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ Multi-Head Attention    │ ❄️ 冻结     │                   │    │
│  │     │  │ (12 heads, dim=768)     │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ Residual Connection     │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ LayerNorm               │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ FFN (MLP)               │ ❄️ 冻结     │                   │    │
│  │     │  │ 768→3072→768            │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ 🔥 Adapter Module       │ 🔥 可训练   │                   │    │
│  │     │  │  - Down: 768→64         │             │                   │    │
│  │     │  │  - ReLU                 │             │                   │    │
│  │     │  │  - Up: 64→768           │             │                   │    │
│  │     │  │  - Residual + Scale     │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     │             ↓                            │                   │    │
│  │     │  ┌─────────────────────────┐             │                   │    │
│  │     │  │ Residual Connection     │             │                   │    │
│  │     │  └──────────┬──────────────┘             │                   │    │
│  │     └─────────────┴──────────────────────────┘                     │    │
│  │                                                                     │    │
│  │  4. Final LayerNorm                                                │    │
│  │     - 输出: [B, 197, 768]                                           │    │
│  │                                                                     │    │
│  │  5. 提取 CLS Token                                                  │    │
│  │     - 输出: [B, 768]                                                │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
└───────────────────────────────────┬─────────────────────────────────────────┘
                                    │
                                    ↓
                          features [B, 768]
                                    │
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│                        Cosine Classifier (fc)                               │
│                        🔥 参与训练和梯度更新                                   │
│  ┌────────────────────────────────────────────────────────────────────┐    │
│  │  CosineLinear(in_features=768, out_features=10)                    │    │
│  │                                                                     │    │
│  │  前向计算:                                                           │    │
│  │  1. 归一化输入: x_norm = x / ||x||                                  │    │
│  │  2. 归一化权重: w_norm = W / ||W||                                  │    │
│  │  3. 计算余弦相似度: logits = σ * (x_norm @ w_norm^T)                │    │
│  │     - σ: 可学习温度参数 (初始化为10)                                 │    │
│  │                                                                     │    │
│  │  参数:                                                              │    │
│  │  - weight: [10, 768]  (10个类别原型)                                │    │
│  │  - sigma: scalar       (温度缩放因子)                                │    │
│  └────────────────────────────────────────────────────────────────────┘    │
└───────────────────────────────────┬─────────────────────────────────────────┘
                                    │
                                    ↓
                            logits [B, 10]
                                    │
                                    ↓
┌─────────────────────────────────────────────────────────────────────────────┐
│                           损失函数计算                                        │
│                                                                             │
│  loss = CrossEntropy(logits, targets)                                      │
│                                                                             │
│  详细:                                                                       │
│  1. Softmax: p = exp(logits) / Σ exp(logits)                               │
│  2. Negative Log-Likelihood: loss = -log(p[target_class])                  │
│                                                                             │
│  示例:                                                                       │
│  - targets: [3, 7, 1, 9, ...]  (batch中每个样本的真实类别)                   │
│  - logits: [B, 10]                                                          │
│  - loss: scalar                                                             │
└───────────────────────────────────┬─────────────────────────────────────────┘
                                    │
                                    ↓
                          ┌─────────────────┐
                          │  反向传播        │
                          │  loss.backward() │
                          └────────┬─────────┘
                                   │
                                   ↓
              ┌────────────────────────────────────────┐
              │  梯度只流向 Adapter 参数                │
              │  (ViT主干被冻结,不接收梯度)              │
              │                                        │
              │  更新参数:                              │
              │  θ_adapter ← θ_adapter - lr * ∇L       │
              │                                        │
              │  优化器: SGD (momentum=0.9)             │
              │  学习率: 0.01 (CosineAnnealing)        │
              │  训练轮数: 80 epochs                    │
              └────────────────────────────────────────┘

┌═══════════════════════════════════════════════════════════════════════════════┐
│                          训练完成后的转换步骤                                   │
└═══════════════════════════════════════════════════════════════════════════════┘

Step 1: construct_dual_branch_network()
  ┌────────────────────────────────────────────────────────────┐
  │ 创建 MultiBranchCosineIncrementalNet_adapt_AC             │
  │                                                            │
  │ 1. 复制原始微调后的ViT模型                                  │
  │    tuned_model = self._network (刚训练完的)                │
  │                                                            │
  │ 2. 加载预训练ViT (不带Adapter,完全冻结)                     │
  │    pretrained_model = get_backbone(args, pretrained=True) │
  │                                                            │
  │ 3. 构建双分支:                                             │
  │    backbones[0] = pretrained_model   (冻结)               │
  │    backbones[1] = tuned_model        (冻结)               │
  │                                                            │
  │ 4. 特征维度变化: 768 → 1536 (768×2)                        │
  └────────────────────────────────────────────────────────────┘

Step 2: update_fc(10, Hidden=15000)
  ┌────────────────────────────────────────────────────────────┐
  │ 创建 ac_model (Analytical Classifier)                      │
  │                                                            │
  │ ac_model = AC_Linear(1536, 15000, 10)                     │
  │                                                            │
  │ 结构:                                                       │
  │ ┌──────────────────────────────┐                           │
  │ │ fc[0]: Linear(1536, 15000)   │ ← ❄️ 随机初始化后立即冻结     │
  │ │        bias=False            │                           │
  │ └──────────┬───────────────────┘                           │
  │            ↓                                               │
  │ ┌──────────────────────────────┐                           │
  │ │ fc[1]: ReLU()                │                           │
  │ └──────────┬───────────────────┘                           │
  │            ↓                                               │
  │ ┌──────────────────────────────┐                           │
  │ │ fc[2]: Linear(15000, 10)     │ ← ❄️ 通过闭式解更新        │
  │ │        bias=False            │                           │
  │ └──────────────────────────────┘                           │
  └────────────────────────────────────────────────────────────┘

Step 3: cls_align(train_loader, model)
  ┌──────────────────────────────────────────────────────────────┐
  │ 使用解析学习 (Analytical Learning) 初始化 ac_model.fc[2]    │
  │                                                              │
  │ 伪代码流程:                                                   │
  │                                                              │
  │ 1. 初始化累积矩阵                                             │
  │    auto_cor = zeros([15000, 15000])  # 自相关矩阵            │
  │    crs_cor = zeros([15000, 10])       # 交叉相关矩阵          │
  │                                                              │
  │ 2. 遍历训练集                                                 │
  │    for (images, labels) in train_loader:                    │
  │        features = model(images)["features"]  # [B, 1536]    │
  │        Φ = ac_model.fc[:2](features)         # [B, 15000]   │
  │        Y = one_hot(labels, 10)                # [B, 10]     │
  │                                                              │
  │        auto_cor += Φ^T @ Φ       # [15000, 15000]           │
  │        crs_cor += Φ^T @ Y        # [15000, 10]              │
  │                                                              │
  │ 3. Ridge Regression优化                                      │
  │    - 候选正则化系数: λ ∈ [1e-8, 1e-7, ..., 1e8]              │
  │    - 通过80/20验证集分割选择最优λ                             │
  │                                                              │
  │ 4. 求解闭式解                                                 │
  │    R = (auto_cor + λ*I)^(-1)     # [15000, 15000]           │
  │    W = R @ crs_cor                # [15000, 10]             │
  │                                                              │
  │ 5. 更新ac_model权重                                           │
  │    ac_model.fc[2].weight = 0.9 * W^T  # [10, 15000]         │
  │    (0.9是经验缩放因子)                                        │
  │                                                              │
  │ 存储:                                                         │
  │    self.R = R  (保存用于后续增量更新)                          │
  └──────────────────────────────────────────────────────────────┘

Step 4: _compute_means()
  ┌──────────────────────────────────────────────────────────────┐
  │ 计算每个类别的原型 (用于后续知识反刍)                          │
  │                                                              │
  │ for class_idx in [0, 1, ..., 9]:                            │
  │     class_samples = 获取类别class_idx的所有训练样本           │
  │     features = model(class_samples)["features"]             │
  │     mean = np.mean(features, axis=0)  # [1536,]             │
  │     cov = np.cov(features.T)          # [1536, 1536]        │
  │     std = sqrt(diag(cov))             # [1536,]             │
  │                                                              │
  │     self._means.append(mean)                                │
  │     self._cov_matrix.append(cov)                            │
  │     self._std_deviations_matrix.append(std)                 │
  └──────────────────────────────────────────────────────────────┘

┌═══════════════════════════════════════════════════════════════════════════════┐
│                        任务0训练完成后的模型状态                                │
│                                                                               │
│  ✅ backbones[0]: 预训练ViT (冻结)                                             │
│  ✅ backbones[1]: 微调后的ViT (冻结)                                           │
│  ✅ fc: CosineLinear(1536, 10) (保留,用于后续增量任务训练)                      │
│  ✅ ac_model: AC_Linear(1536, 15000, 10) (冻结,用于推理)                       │
│  ✅ R: [15000, 15000] 逆矩阵 (用于后续RLS更新)                                  │
│  ✅ _means: 10个类别原型向量                                                    │
│  ✅ _old_network: None (第一个任务没有旧网络)                                   │
└═══════════════════════════════════════════════════════════════════════════════┘
```

## 3️⃣ **任务 t (t≥1) - 训练阶段 (以任务1为例)**

```
│  │ 输入: images │   │ 输入: images │   │    输入: images          │       │

Step 1: _compute_means() - 计算新类原型
  ┌──────────────────────────────────────────────────────────────┐
  │ 对新增类别10-19,计算每个类的统计量                            │
  │                                                              │
  │ for class_idx in [10, 11, ..., 19]:                         │
  │     samples = 获取类别class_idx的训练样本                     │
  │     features = network(samples)["features"]  # [N, 2304]    │
  │                                                              │
  │     mean = np.mean(features, axis=0)         # [2304,]      │
  │     cov = np.cov(features.T)                 # [2304, 2304] │
  │     std = np.sqrt(np.diag(cov))              # [2304,]      │
  │                                                              │
  │     self._means.append(mean)                                │
  │     self._cov_matrix.append(cov)                            │
  │     self._std_deviations_matrix.append(std)                 │
  │                                                              │
  │ 现在: _means包含20个类别的原型 (0-19)                         │
  └──────────────────────────────────────────────────────────────┘

Step 2: IL_align() - 增量更新 ac_model.fc[2]
  ┌──────────────────────────────────────────────────────────────┐
  │ 使用递归最小二乘 (RLS) 更新 ac_model                          │
  │                                                              │
  │ 初始化:                                                       │
  │   W = ac_model.fc[2].weight.T  # [15000, 20], 前10列已有值   │
  │   R = self.R                    # [15000, 15000], 从任务0保存│
  │                                                              │
  │ for (images, labels) in train_loader:                       │
  │     features = network(images)["features"]    # [B, 2304]   │
  │     Φ = ac_model.fc[:2](features)             # [B, 15000]  │
  │     Y = one_hot(labels, 20)                    # [B, 20]    │
  │                                                              │
  │     # Sherman-Morrison-Woodbury公式                          │
  │     R = R - R @ Φ^T @ (I + Φ @ R @ Φ^T)^(-1) @ Φ @ R       │
  │     W = W + R @ Φ^T @ (Y - Φ @ W)                           │
  │                                                              │
  │ 更新权重:                                                     │
  │   ac_model.fc[2].weight = W.T                               │
  │   self.R = R  (保存用于下一任务)                              │
  │                                                              │
  │ 复杂度分析:                                                   │
  │   - 每个batch: O(B × 15000^2) (矩阵运算)                     │
  │   - 比从头计算逆矩阵快得多                                     │
  └──────────────────────────────────────────────────────────────┘

Step 3: 知识反刍机制 (如果 cali_weight > 0)
  ┌──────────────────────────────────────────────────────────────┐
  │ 3a. _compute_relations() - 计算新旧类关系                     │
  │                                                              │
  │ old_means = _means[0:10]      # [10, 2304]                  │
  │ new_means = _means[10:20]     # [10, 2304]                  │
  │                                                              │
  │ # 归一化后计算余弦相似度矩阵                                   │
  │ similarity = normalize(old_means) @ normalize(new_means).T  │
  │ # [10, 10]                                                  │
  │                                                              │
  │ # 为每个旧类找到最相似的新类                                   │
  │ relations = argmax(similarity, axis=1) + 10                 │
  │ # [10,] 例如: [12, 15, 18, 10, 19, 13, 11, 17, 14, 16]      │
  │                                                              │
  │ 解释: 旧类0最相似新类12, 旧类1最相似新类15, ...               │
  └───────────────────────────────────────��──────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │ 3b. _build_feature_set() - 构建伪特征集                       │
  │                                                              │
  │ vectors_train = []                                          │
  │ labels_train = []                                           │
  │                                                              │
  │ # 收集新类的真实特征                                          │
  │ for class_idx in [10, 11, ..., 19]:                         │
  │     samples = 获取类别class_idx的训练样本                     │
  │     features = network(samples)["features"]                 │
  │     vectors_train.append(features)                          │
  │     labels_train.append([class_idx] * len(features))        │
  │                                                              │
  │ # 为旧类生成伪特征                                            │
  │ for old_class in [0, 1, ..., 9]:                            │
  │     related_new = relations[old_class]  # 最相似的新类       │
  │                                                              │
  │     # 新类特征 - 新类均值 + 旧类均值 = 旧类伪���征              │
  │     pseudo_features = (                                     │
  │         vectors_train[related_new - 10]                     │
  │         - _means[related_new]                               │
  │         + _means[old_class]                                 │
  │     )                                                       │
  │                                                              │
  │     vectors_train.append(pseudo_features)                   │
  │     labels_train.append([old_class] * len(pseudo_features)) │
  │                                                              │
  │ # 创建DataLoader                                             │
  │ _feature_trainset = DataLoader(                             │
  │     FeatureDataset(vectors_train, labels_train),            │
  │     batch_size=48, shuffle=True                             │
  │ )                                                           │
  │                                                              │
  │ 数据量:                                                       │
  │ - 新类真实特征: 10类 × ~500样本/类 = ~5000样本                │
  │ - 旧类伪特征: 10类 × ~500样本/类 = ~5000样本                  │
  │ - 总计: ~10000个特征向量用于反刍                              │
  └──────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────┐
  │ 3c. cali_weight() - 基于错误预测校准权重                      │
  │                                                              │
  │ 初始化:                                                       │
  │   W = ac_model.fc[2].weight.T                               │
  │   R = self.R                                                │
  │                                                              │
  │ for (features, labels) in _feature_trainset:                │
  │     Φ = ac_model.fc[:2](features)         # [B, 15000]      │
  │     Y = one_hot(labels, 20)                # [B, 20]        │
  │                                                              │
  │     # 只对预测错误的样本进行更新                               │
  │     logits = ac_model.fc[2](Φ)                              │
  │     preds = argmax(logits, dim=1)                           │
  │     wrong_mask = (preds != labels)                          │
  │                                                              │
  │     Φ_wrong = Φ[wrong_mask]               # [B', 15000]     │
  │     Y_wrong = Y[wrong_mask]                # [B', 20]       │
  │                                                              │
  │     if len(Φ_wrong) > 0:                                    │
  │         # RLS更新                                            │
  │         R = R - R@Φ_wrong.T@inv(I+Φ_wrong@R@Φ_wrong.T)@... │
  │         W = W + R @ Φ_wrong.T @ (Y_wrong - Φ_wrong @ W)    │
  │                                                              │
  │ 更新: ac_model.fc[2].weight = W.T                           │
  │                                                              │
  │ 效果:                                                         │
  │ - 重点强化模型在旧类上的表现                                   │
  │ - 只修正错误,不影响正确预测                                    │
  │ - 缓解灾难性遗忘                                              │
  └──────────────────────────────────────────────────────────────┘

Step 4: after_task() - 保存当前模型状态
  ┌──────────────────────────────────────────────────────────────┐
  │ self._old_network = self._network.copy().freeze()           │
  │                                                              │
  │ 作用: 保存任务1的完整模型状态,用于任务2的知识蒸馏              │
  └──────────────────────────────────────────────────────────────┘
```

## 🔬 训练过程详解

### 整体流程图

```
开始
  ↓
任务 0 (初始任务)
  ├─ 加载预训练 ViT + Adapter
  ├─ 微调适配器 (tuned_epoch=80)
  ├─ 构建双分支网络 (Cosine + AC)
  ├─ 初始化 Analytical Learning 分类头
  ├─ 计算类原型均值
  └─ cls_align: 使用最小二乘求解初始权重
  ↓
任务 1-N (增量任务)
  ├─ 扩展分类头到新类别数
  ├─ 冻结 AC 模型
  ├─ 训练适配器 (progreesive_epoch=80)
  │   ├─ Cross-Entropy Loss
  │   ├─ Knowledge Distillation Loss (如果 lambda_fkd > 0)
  │   ├─ Parameter Regularization (如果 rg > 0)
  │   └─ EMA 更新 (如果 alpha > 0)
  ├─ 计算新类原型均值
  ├─ IL_align: 增量更新 AL 权重
  ├─ (可选) 知识反刍机制:
  │   ├─ cali_prototye_model: 校准原型模型
  │   ├─ _compute_relations: 计算新旧类关系
  │   ├─ _build_feature_set: 构建反刍特征集
  │   └─ cali_weight: 基于错误预测校准权重
  └─ 评估当前任务性能
  ↓
结束：输出所有任务的平均准确率
```

### 第一个任务 (Task 0) 详细流程

**目标**: 在初始类别上微调预训练模型，建立基础知识

1. **网络初始化**

   ```python
   # 加载预训练 ViT-B/16 + Adapter
   self._network = SimpleVitNet_AL(args, True)
   # 初始化 Cosine 分类器
   self._network.fc = CosineLinear(feature_dim=768, num_classes=init_cls)
   ```

2. **适配器微调** (`_init_train`)

   - 训练轮数: `tuned_epoch` (默认80)
   - 损失函数: Cross-Entropy
   - 优化器: SGD (momentum=0.9)
   - 学习率调度: CosineAnnealing
   - 只训练 Adapter 参数，冻结 ViT backbone

3. **构建双分支网络** (`construct_dual_branch_network`)

   ```python
   # 创建两个分支:
   # Branch 1: Cosine Classifier (用于训练)
   # Branch 2: Analytical Classifier (用于推理)
   network = MultiBranchCosineIncrementalNet_adapt_AC(args, True)
   ```

4. **初始化 Analytical Learning 分类头** (`cls_align`)

   - 提取训练集特征
   - 计算自相关矩阵: `auto_cor = Φ^T Φ`
   - 计算交叉相关矩阵: `crs_cor = Φ^T Y`
   - Ridge regression 优化: 选择最佳正则化参数 λ
   - 求解最小二乘: `W = (Φ^T Φ + λI)^(-1) Φ^T Y`

5. **计算类原型** (`_compute_means`)

   - 为每个类别计算特征均值
   - 计算协方差矩阵和标准差（用于后续知识反刍）

### 增量任务 (Task 1-N) 详细流程

**目标**: 学习新类别，同时保持对旧类别的记忆

1. **网络扩展**

   ```python
   # 扩展分类头以包含新类别
   self._network.update_fc(total_classes, Hidden, cosine_fc=True)  # Cosine分支
   self._network.update_fc(total_classes, Hidden)  # AC分支
   # 冻结 AC 模型参数
   for param in self._network.ac_model.parameters():
       param.requires_grad = False
   ```

2. **适配器增量训练** (`_progreessive_train`)

   - 训练轮数: `progreesive_epoch` (默认80)
   - 创建 EMA 模型用于动量更新

   **损失函数组成**:

   ```python
   # 基础分类损失
   loss_ce = CrossEntropy(logits, targets)
   
   # (可选) 知识蒸馏损失 (lambda_fkd > 0)
   loss_kd = KL_Divergence(
       current_logits[:, :old_classes] / T,
       old_logits[:, :old_classes] / T
   ) * T^2
   
   # (可选) 参数正则化 (rg > 0)
   loss_reg = ||θ_new - θ_old||^2
   
   # 总损失
   loss = loss_ce + λ_fkd * loss_kd + rg * loss_reg
   ```

3. **EMA 动量更新** (每个 epoch 后)

   ```python
   # 如果 alpha > 0
   θ_ema = alpha * θ_ema + (1 - alpha) * θ_current
   # 训练结束后将 EMA 参数复制回主模型
   ```

4. **计算新类原型** (`_compute_means`)

   - 对新增的每个类别计算特征均值和统计量

5. **增量 AL 权重更新** (`IL_align`)

   - 使用递归最小二乘（RLS）算法

   - 对每个新样本批次:

     ```python
     # Sherman-Morrison-Woodbury 公式
     R_new = R_old - R_old @ Φ^T @ (I + Φ @ R_old @ Φ^T)^(-1) @ Φ @ R_old
     W_new = W_old + R_new @ Φ^T @ (Y - Φ @ W_old)
     ```

6. **知识反刍机制** (如果 `cali_weight > 0`)

   a. **校准原型模型** (`cali_prototye_model`)

      - 为旧类创建校准模型

   b. **计算新旧类关系** (`_compute_relations`)

      ```python
   # 找到每个旧类最相似的新类
   similarity = normalize(old_means) @ normalize(new_means)^T
   relations[i] = argmax(similarity[i]) + known_classes
      ```

   c. **构建反刍特征集** (`_build_feature_set`)

      - 收集新类的真实特征

      - 为旧类生成伪特征:

        ```python
        pseudo_feature[old_class] = 
            feature[related_new_class] - mean[related_new_class] + mean[old_class]
        ```

   d. **基于错误校准权重** (`cali_weight`)

      - 只对预测错误的样本进行权重更新
      - 再次应用 RLS 算法

7. **任务后处理** (`after_task`)

   - 保存当前网络为旧网络 `_old_network`
   - 用于下一任务的知识蒸馏

## ❓ 疑问点和建议

### 关键疑问

1. **预训练权重路径硬编码**

   - **问题**: `backbone/vision_transformer_adapter.py` 中权重路径需手动修改
   - **建议**: 将权重路径作为配置参数，在 JSON 文件中指定

   ```json
   {
       "pretrained_weight_path": "checkpoints/dino_vitbase16_pretrain.pth"
   }
   ```

2. **数据集路径配置不灵活**

   - **问题**: `utils/data.py` 中数据路径硬编码
   - **建议**: 支持环境变量或配置文件指定

   ```python
   data_path = os.environ.get('MOAL_DATA_PATH', args.get('data_path', '/default/path'))
   ```

3. **记忆机制实现但未使用**

   - **观察**: 配置中 `memory_size=0`，但代码包含 exemplar 构建逻辑
   - **疑问**: MoAL 是否完全不需要样本存储？还是可选功能？
   - **建议**: 文档说明是否支持 memory-based 变体

4. **多 GPU 训练稳定性**

   - **问题**: 使用 `nn.DataParallel`，可能在大规模实验中效率不高
   - **建议**: 考虑升级到 `DistributedDataParallel`

5. **评估指标**

   - **观察**: 同时输出 CNN 和 NME 准确率
   - **疑问**: 最终报告应该使用哪个指标？
   - **建议**: 文档明确说明 NME (Analytical Learning) 是主要指标

6. **知识反刍的计算开销**

   - **观察**: `_build_feature_set` 需要遍历所有新类样本
   - **疑问**: 在大规模数据集上是否有性能瓶颈？
   - **建议**: 考虑采样策略或批处理优化

7. **Ridge regression 超参数选择**

   - **观察**: `optimise_ridge_parameter` 使用固定的候选集
   - **建议**: 可以根据数据集规模动态调整搜索范围

### 使用建议

1. **首次运行建议**
   - 先在 CIFAR-100 上验证环境配置正确
   - 使用较小的 Hidden 维度（如 10000）快速测试
   - 确认日志输出和准确率符合预期

2. **超参数调优建议**
   - `alpha`: 大数据集建议 0.999，小数据集可以降低到 0.99
   - `Hidden`: 根据 GPU 显存调整（15000需要约10-12GB显存）
   - `lambda_fkd`: 任务数较多时建议增大（0.5-1.0）

3. **硬件配置建议**
   - CIFAR-100: 单卡 1080Ti/2080Ti (11GB) 即可
   - ImageNet-R/A: 建议 3090/V100 (24GB)
   - OmniBenchmark: 建议 A100 (40GB) 或多卡并行

4. **实验时间估计**
   - CIFAR-100 (10 tasks): ~8-10 小时 (单卡 3090)
   - ImageNet-R (10 tasks): ~20-24 小时 (单卡 3090)
   - 主要时间消耗在适配器微调阶段

## 🔧 常见问题排查

### 1. CUDA Out of Memory

**解决方案**:

- 减小 `batch_size` (48 → 24)
- 减小 `Hidden` 维度 (15000 → 10000)
- 使用梯度累积

### 2. 数据集加载失败

**检查项**:

- `utils/data.py` 中路径配置是否正确
- 数据集目录结构是否符合要求（train/, val/）
- CIFAR-100 需要网络连接以自动下载

### 3. 预训练权重加载失败

**检查项**:

- 权重文件是否下载完整
- `checkpoints/` 目录是否存在
- `vision_transformer_adapter.py` 中路径配置是否正确

### 4. 准确率异常低

**可能原因**:

- 预训练权重未正确加载
- 学习率设置不当
- 数据增强过强或过弱
- Ridge regression 参数选择失败

### 5. 训练过程卡住

**可能原因**:

- `num_workers` 设置过高，尝试减少到 4 或 0
- 数据加载瓶颈，检查磁盘 I/O

#### ema更新解释：

~~~~
┌═══════════════════════════════════════════════════════════════════════┐
│                  EMA 更新的完整时间线                                   │
└═══════════════════════════════════════════════════════════════════════┘

任务1训练开始:
  ↓
创建 EMA_model = network.copy().freeze()
  ├─ EMA_model.backbones[2] = 复制自 network.backbones[2]
  │  └─ Adapter参数初始化 (从Backbone[1]继承)
  │
  └─ 冻结 EMA_model 所有参数 (requires_grad=False)
  ↓

Epoch 1:
  ├─ 训练 network.backbones[2].Adapter (梯度下降)
  │  └─ 参数从 θ₀ 变为 θ₁
  │
  └─ EMA更新 EMA_model.backbones[2].Adapter
     └─ θ_ema₁ = 0.999*θ₀ + 0.001*θ₁
  ↓

Epoch 2:
  ├─ 训练 network (θ₁ → θ₂)
  └─ EMA更新: θ_ema₂ = 0.999*θ_ema₁ + 0.001*θ₂
  ↓

...
  ↓

Epoch 80:
  ├─ 训练 network (θ₇₉ → θ₈₀)
  └─ EMA更新: θ_ema₈₀ = 0.999*θ_ema₇₉ + 0.001*θ₈₀
  ↓

训练结束后:
  └─ 复制 EMA 参数回主模型
     network.backbones[2].Adapter.params ← EMA_model.backbones[2].Adapter.params
  ↓

最终模型状态:
  └─ network.backbones[2] 使用的是平滑后的 Adapter 参数
     (而非最后一次训练的参数)
~~~~
